# Task Manager Restful App

### Task description:  
Develop a task management application using back-end tecnologies to pratice Java and some frameworks/patterns.

### Tecnologies used:

    - Spring Boot
    - Maven
    - Java 11
    - Spring Web
    - H2 Database
    - Swagger UI
    - JUnit

### Application Description:

    This Restful app has CRUD operations made by endpoints and some unity tests.
